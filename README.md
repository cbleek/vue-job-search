# Yavue

yavue is a vue frontend for the job search in a YAWIK installation. A Google API Key and a Yawik Endpoint can 
be configured in .env or defined as ENV vars.

Demo:

http://yavue.de

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
